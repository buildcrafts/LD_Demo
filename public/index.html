3. Remediate

You can release or roll back the flag from the LaunchDarkly UI, with a cURL command, or with the optional in-app control buttons below if you add an API token on the server.

This log makes the demo easier to explain during the walkthrough. You will see when the SDK initializes, when contexts change, and when the flag updates in real time.
